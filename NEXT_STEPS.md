### 3. Quick DNS Setup
When you add the domain, Vercel will show you exactly what DNS records to add. Generally, you'll need:

**For the root domain (aarongreenberg.net):**
```
Type: A
Name: @
Value: 76.76.21.21
```

**For www redirect:**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
```
